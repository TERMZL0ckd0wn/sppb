## Mengenai SPPB AL-GAZEL

SPPB AL-GAZEL adalah sebuah sistem pengurusan perpustakaan digital yang direka untuk memudahkan urusan harian di Pusat Sumber Al-Gazel, Kolej Vokasional Datuk Seri Abu Zahar Isnin. Sistem ini memfokuskan kepada kepantasan transaksi menggunakan teknologi pengimbas kod bar, mengurangkan ralat manusia, dan mendigitalkan rekod yang sebelum ini diuruskan secara manual.

Sistem ini meringankan beban tugas pengawas pusat sumber melalui fungsi automatik seperti:

- Transaksi Pintar: Peminjaman dan pemulangan buku menggunakan kod bar/QR.
- Pangkalan Data Berpusat: Pengurusan maklumat ahli (pelajar/pensyarah) dan inventori buku secara real-time.
- Carian Efisien: Modul carian tajuk dan status ketersediaan buku yang pantas.
- Laporan Digital: Pemantauan senarai tunggakan peminjaman yang lebih sistematik.

## Teknologi Utama

Kami menggunakan gabungan teknologi yang stabil dan efisien untuk memastikan kelancaran sistem:

- [PHP](https://www.php.net/) - Enjin skrip sebelah pelayan.
- [MySQL](https://www.mysql.com/) - Sistem pengurusan pangkalan data hubungan.
- [JavaScript](https://www.javascript.com/) - Interaksi antaramuka dinamik.
- [Tailwind CSS](https://tailwindcss.com/) - Reka bentuk antaramuka moden dan responsif.

## Metodologi & Analisis

Pembangunan sistem ini berasaskan kitaran hayat pembangunan sistem (SDLC) yang merangkumi fasa perancangan, analisis (ERD & Context Diagram), reka bentuk (Wireframe), dan pembangunan kod. Berdasarkan analisis Bab 4, sistem ini mencapai Skor Min Keseluruhan 3.88, menandakan tahap kepuasan pengguna yang sangat tinggi.

## Cara Penggunaan

Langkah-langkah untuk menjalankan projek ini di persekitaran tempatan anda:

- Klon repositori: git clone https://github.com/Ice0608/sppb.git
- Pastikan perisian XAMPP telah dipasang dan diaktifkan (Apache & MySQL).
- Letakkan folder projek ke dalam direktori htdocs.
- Import fail SQL yang disediakan ke dalam phpMyAdmin.
- Akses sistem melalui pelayar web di alamat "localhost/sppb/public" atau menggunakan terminal pada folder tersebut dengan menggunakan "php artisan serve".
